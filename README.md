## How to test locally

### Installation
Install jekyll with instructions [here](https://jekyllrb.com/docs/installation/macos/).

``` bash
# Finally
gem install jekyll bundler
bundle install
```
### Running locally
``` bash
bundle exec jekyll serve
```
